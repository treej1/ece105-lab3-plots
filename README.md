# Sensor Plot Generation (ECE105 Lab 3)

## Overview

This repository contains generate_plots.py, a small script that generates synthetic temperature sensor data and produces publication-quality visualizations (scatter, histogram, and box plot) saved as a single PNG file.

## Requirements

- Anaconda/Miniconda (recommended)
- Activate the provided environment named `ece105` before installing packages.

## Installation

1. Activate the environment:

```powershell
conda activate ece105
```

2. Install required packages (using conda or mamba):

```powershell
conda install numpy matplotlib
# or
mamba install numpy matplotlib
```

## Usage

Run the script from the repository root:

```powershell
python generate_plots.py
```

The script uses a fixed RNG seed (seed = 1234) to create reproducible synthetic data and saves the figure to disk.

## Output files

- `sensor_analysis.png` — a single figure (1x3 subplots) containing:
  - Scatter plot of Sensor A and Sensor B vs time
  - Overlaid histogram of both sensors with mean lines
  - Side-by-side box plots with an overall mean line

## Files

- `generate_plots.py` — script that generates the data and draws the plots.
- `sensor_analysis.png` — generated by running the script.
